# whistle

> 提供给uTools的Whistle插件，方便操作Whistle规则

项目地址：[utools-whistle](https://github.com/MrLeo/utools-whistle)

关于我：[xuebin.me](https://xuebin.me)

# Features

- [x] 为手机 SS 代理软件生成 Whistle 二维码
- [x] Whistle 规则开关
- [x] 检查并初始化 Whistle 环境
- [x] 获取 Whistle 状态
- [x] 启动/关闭/刷新 Whistle
